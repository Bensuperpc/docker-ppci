#### Install
You need Linux distribution like Ubuntu or Manjaoro

```sh
git clone https://github.com/Bensuperpc/docker-docker-ppci.git
```
```sh
cd docker-ppci
```
#### Usage

```sh
./sjasmplus.sh ppci -h
```
#### Build
```sh
make linux/amd64 or make linux/arm64 or make linux/arm/v7
```
